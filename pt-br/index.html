Sobre mim

Meu nome é Vinícius Flesch Kern, tenho 17 anos, sou aluno do Ensino Médio que adora programação, robôs e natureza!

 Já se passaram 5 anos desde que executei meu primeiro "Hello World". No início, fiz um pouco de programação em Lua no Roblox, mas nunca me dediquei muito à isso, depois comecei a fazer alguns cursos de desenvolvimento web que encontrei online por dois ou três anos, até me achar capaz de seguir em frente para alguns tópicos mais avançados, como back-end.

 No começo fiz alguns cursos de PHP, mas para ser sincero, não gostei muito então decidi seguir em frente, foi quando encontrei o Node.js que gostei por algum tempo, mas acabei decidindo mudar de linguagem mais uma vez. então novamente, algum dia, encontrei alguns cursos sobre Java de um instituto de tecnologia aqui do meu estado, e como Minecraft foi um dos primeiros jogos de computador que joguei, sempre tive um pouco de curiosidade por Java, então tentei e pronto: encontrei minha linguagem preferida para desenvolvimento de back-end. Além disso também aprendi alguns bancos de dados como MySQL e PostrgreSQL, pois agora precisava armazenar dados para alguns de meus aplicativos.

 Como mencionei que gosto de robôs, é preciso dizer que nos intervalos que tive no desenvolvimento web, aprendi a programar em Arduino, o que foi mais uma experiência maravilhosa: ver as coisas que você construiu ganhando vida bem na sua frente e você ser capaz de interagir com elas da maneira que projetou, em vez de apenas vê-las por uma tela. Além com isso também fiz alguns projetos com o Esp32 devido à sua compatibilidade com a linguagem do Arduino e especificações muito mais poderosas em comparação com o Arduino Uno que eu tinha, o que levou a alguns projetos mais avançados.

 Em relação à natureza, sempre adorei acampar e ficar ao ar livre, por isso aos 10 anos entrei no Grupo Escoteiro de minha cidade onde aprendi muito sobre ética e como me comportar em situações estressantes.

Stack de Trabalho

HTML

Capacidade de estruturar páginas de forma correta e eficiente, tornando-as facilmente estilizáveis através de CSS, propositais, e legíveis, fazendo uso de boas práticas, garantindo pequenos tempos de carregamento, além de acessibilidade.

CSS

Design dinâmico de páginas de acordo com as proporções da tela, aliado ao uso eficiente de classes e organização de arquivos, conferindo às páginas uma ótima aparência em qualquer dispositivo.

Javascript

Escrever código otimizado do lado do cliente para todos os propósitos, desde mais métodos de estilização (como paralaxe), até aplicativos web de cliente completos e integração segura de back-end via http.

Java

Programação de código Java altamente sustentável, fazendo uso eficiente de polimorfismo e código legível. Além de escrever código back-end seguro para servidores, evitando diversos tipos de ataques.

Spring

Uso do framework Spring para criação de APIs RESTFul resilientes para integração com aplicativos web.

Docker

Criação e Dockerização de apps para implantação rápida e fácil na nuvem, proporcionando alta disponibilidade de serviços e escalabilidade sob demanda, além de evitar resultados inesperados durante a execução em diferentes máquinas.

PostgreSQL

Gerenciando dados de forma segura e rápida, permitindo ótima integração com Docker.

MySQL

Gerenciando dados no sistema de gerenciamento de banco de dados SQL mais utilizado.

Arduino

Criação de projetos de hardware customizados baseados em unidades de microcontroladores como Arduino e Esp criando também firmware otimizado, possibilitando sistemas automatizados e interfaces customizadas para diversos fins, bem como projetos de Internet das Coisas, para comunicação dispositivo a dispositivo de longo alcance.
